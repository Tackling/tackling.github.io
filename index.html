Twitch Bot Commands

Join / Leave:

?Join - Join the user's channel.

?Part - Leave the user's channel.

Entertainment:

?8ball [input] - Play a game of 8-ball with a user.

?Coinflip [choice] - Perform a coin flip.

?Dice [choice] - Roll a dice.

?Joke - Tell a random joke.

?Kanye - Share a random Kanye West quote.

?Fact - Share a random fact.

?Rps [choice] - Play a game of Rock-Paper-Scissors.

User Information:

?Followage - Show how long a user has been following the streamer.

?Followage [username] - Show how long a user has been following the user.

?Subage - Show how you have been subbed to a streamer.

?Subage [username] - Show how long a user has been subbed to someone.

?Userid [userID] - Convert a user ID to a username.

?Iduser [username] - Convert a username to user ID.

?Accountage - Displays the account age of your Twitch account.

?Accountage [username] - Displays the account age of a Twitch user.

?Connectedto [username] - Displays the number of channels a user is connected to.

?Following - Displays the channels you are following.

?Following [username] - Display the channels a user is following.

?Mod [username] - Display the number of channels a user is moderating.

?Pfp [username] - Get the profile picture of a user.

?Banned [username] - Checks a ban status of a user.

?Baninfo [username] - Checks the ban history for a user (partners only).

?User [username] - Display alot of info about a user.

Channel Information:

?Chatters - Display the number of people in chat.

?Followers - Display the follower count of the streamer.

?Followers [username] - Display the follower count of the user.

?Founders - Show who has the founder's badge of a channel.

?Mods - Display a list of mods for a streamer.

?Vip - Display a list of vips for a streamer.

?Tags - Display the tags of a stream.

?Viewcount - Display the view count of a streamer.

?Game - Displays the game the streamer is currently playing.

Interaction:

?Hug [username] - Give someone in chat a virtual hug.

?Welcome [username] - Send a welcome message to a user.

?Compliment [username] - Compliment a user.

?Gpt [text] - Generates AI-powered text based on input.

Utility:

?Commands - List all available bot commands.

?Language [text] - Identify the language of a word/sentence.

?Translate [text] - Translate a message to English.

?Weather [location] - Get the current weather conditions for a location you specify.

?Time [location] - Show the time in a location you provide.

?Title - Display the current title of a streamer.

?Uptime - Show how long a streamer has been live.

?Offline - Show how long it's been since a streamer has been live.

?Ping - Ping the bot.

?Channels - Displays the number of joined channels and channels where the bot is a moderator.

?Urban [text] - Look up a definition for a word using Urban Dictionary.

?Twitchstats - Displays the total number of accounts on Twitch.

?Checkuser [username] - Checks if a Twitch username is available or not.

?Morsecode [text] - Converts text to Morse code.

?Decodemorse [morsecode] - Converts Morse code to text.

?Owner - Displays information about the bot's owner.

Bot Control (Admin-Only):

?Rejoin [username] - Rejoin a user's channel.

?Rejoin all - Rejoin all user's channel.

?Reloadcmds - Reload the command list.

?Shutdown - Shut the bot down.
