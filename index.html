TacklingBOT Commands

TacklingBOT is a versatile Twitch chat bot with a wide range of commands, from entertainment and user information to channel insights and utility tools. Engage your viewers, manage your stream, and keep your community entertained with this all-in-one bot.

Join / Leave:

Command Name Aliases Cooldown Description
?Join 1 Second Join the user's channel.
?Part 3 Seconds Leave the user's channel.

Entertainment:

Command Name Aliases Cooldown Description
?8ball [input] 2 Seconds The bot gives an 8-ball response.
?Coinflip [choice] 4 Seconds Perform a coin flip.
?Dice [choice] 4 Seconds Roll a dice.
?Joke 4 Seconds Tell a random joke.
?Kanye ye 4 Seconds Share a random Kanye West quote.
?Fact 4 Seconds Share a random fact.
?Rps [choice] 4 Seconds Play a game of Rock-Paper-Scissors.

User Information:

Command Name Aliases Cooldown Description
?Followage 4 Seconds Show how long a user has been following the streamer.

?Subage 4 Seconds Show how you have been subbed to a streamer.

?Userid [username] 4 Seconds Convert a username to user ID.
?Iduser [username] 4 Seconds Convert a user ID to username.
?Accountage 4 Seconds Displays the account age of your Twitch account.
?Accountage [username] 4 Seconds Displays the account age of a Twitch user.
?Connectedto [username] 4 Seconds Displays the number of channels a user is connected to.
?Following 4 Seconds Displays the channels you are following.
?Following [username] 4 Seconds Displays the channels a user is following.
?Mod [username] 4 Seconds Displays the number of tracked channels a user is moderating.
?Pfp [username] 4 Seconds Get the profile picture of a user.
?Banned [username] 4 Seconds Checks a ban status of a user.
?Baninfo [username] 4 Seconds Checks the ban history for a user (partners only).
?User [username] 4 Seconds Display info about a user.

Bot Control (Admin-Only):

Command Name Aliases Cooldown Description
?Devjoin [username] 1 Seconds Join a user's channel.
?Devjoin all 1 Seconds Join all user's channel.
?Reloadcmds rc 4 Seconds Reload the command list.
?Shutdown 4 Seconds Shut the bot down.
?Blacklist add [userid] 1 Second Blacklist a user from using commands.
?Blacklist remove [userid] 1 Second Remove a user from being blacklisted.
